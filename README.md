# mobile app for university project:
TYar - IT Equipment Advisor
ITYar is a practical and intelligent mobile application developed to help Iranian users and businesses make better and more informed decisions when purchasing IT equipment.
What ITYar Offers:
-Network Equipment Advisor: Recommends complete network solutions (routers, switches, cables, firewalls, patch panels, UPS, access points, NAS, etc.) based on the number of users, building floors, area, outdoor cabling needs, security requirements, and budget.
-Custom PC Builder: Helps users build a desktop computer tailored to their specific needs — whether for daily use, programming, graphic design, or professional gaming  with detailed component suggestions and total cost estimation.
-Laptop Recommender: Suggests the most suitable laptops according to usage, preferred brand, and available budget.
